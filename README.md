# Project Stockio (or Stock.io)

- 주요 stock 정보를 중계하는 경량 백엔드 서비스
- 증권사 API(예: kiwoom REST API)의 google spreadsheet importxml() bridge 서비스
- see `docs/*.md` for more details.

## 기존 작업, 목표 수정 후 작업 방식 변경
- 개인적으로 직접 API 분석, 설계, 구현 방식으로 진행하던 작업은 너무 지지부진하여 중단함.
- 기존 작업 결과물은 poc/ 아래로 모두 이동 처리함.
- 새로운 활용처를 발견하여 목표를 새로 작성하고 기존 poc/ 결과물을 활용하여 agentic 방식으로 개발.
