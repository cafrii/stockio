# CLAUDE.md

- 이 파일은 Claude Code(claude.ai/code)가 이 저장소의 문서를 작업할 때 지침을 제공한다.
- 먼저 공통 지침은 전역 설정을 참고한다. ( ~/.ai/rules.md )
- 프로젝트 지침은 이 리포지토리의 docs/project.md 에 기술되어 있다.
- 전역 지침과 프로젝트 지침이 충돌한 경우에는 프로젝트 지침이 우선한다.

# 작업 순서
- 맥락 파악. 단계 목표 파악. 
- TODO 를 '진행중' 으로 변경.
- 문제 해결 (조사, 분석, 설계, 코딩, 단위시험, E2E 시험 등 뭐가 되었든..)
- 사용자 시험을 위해 일부 문서화. (진행 내용, 시험 절차 가이드 등)
- 사용자가 진행된 내용을 리뷰, 시험. 사용자와의 질문/답변 대화.
- 모든 것이 해결되면 사용자의 확인 하에 단계 완료. (status.md 업데이트)
- "사용자"가 git commit 후 다음 단계 진행 선언.

# 참고 지침
- 대화를 이끌어가는 나의 보유 기술 및 환경에 대한 정보는 ~/.ai/profile.md 를 참고한다.

# 특이사항
- git commit 은 사용자가 수행한다.
